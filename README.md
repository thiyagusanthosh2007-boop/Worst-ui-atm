# Worst UI ATM Interface

This project was created for the AI Olympics Round 4: Worst UI Challenge.

## Objective
To intentionally design a confusing ATM interface that violates UI/UX design principles.

## Features
- Misleading buttons
- Excessive colors
- Random popups
- Cluttered layout
- Confusing navigation

## Demo
Users experience frustration while interacting with the interface, demonstrating how poor UI design impacts usability.

## Technologies Used
- HTML
- CSS
- JavaScript
